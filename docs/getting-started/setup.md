# OhMyUniversity! - Setup

This project is organized as a multi-repository ecosystem.

Each repository corresponds to a specific application or service:

- `<project>-web` > Web application
- `<project>-mobile` > Mobile application
- `<project>-desktop` > Desktop application
- `<project>-api` > Backend and data layer

### How to explore the project

1. Clone the relevant repository
2. Review its `architecture/` folder
3. Start from the `README.md` in each section
4. Follow the documentation structure inside the repo

No global monorepo setup is required.
