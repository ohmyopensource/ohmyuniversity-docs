# OhMyUniversity! - Getting Started

This section provides a quick introduction to the OhMyUniversity! project.

It is recommended to read the files in the following order:

1. `overview.md` - what the project is and what it does
2. `setup.md` - how to navigate the repository structure
3. `api.md` - how data is exposed and used
4. `contribute.md` - how to contribute to the project

This guide is intended as a lightweight entry point to understand the ecosystem.
