# OhMyUniversity! - Overview

OhMyUniversity! is an open source project that aggregates, normalizes, and exposes data from the Italian and European university ecosystem.

The main goal is to unify fragmented educational data and make it easily accessible through a consistent interface.

### Main components

- **Web application** (`<project>-web`)
- **Mobile application** (`<project>-mobile`)
- **Desktop application** (`<project>-desktop`)
- **Backend API** (`<project>-api`)
- **External data sources** (MIM, CINECA, European Data Portal)

### Data flow

External sources > Backend API > Client applications
