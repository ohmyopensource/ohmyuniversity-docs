# 3.3.1 Usability

The usability requirements define how easily users must be able to interact with **OhMyUniversity!** when accessing information, completing tasks, and navigating the platform. Since the system is intended to simplify the university digital experience, the interface must be efficient, accessible, and easy to learn.

- **Navigation Efficiency:** An expert user must be able to access critical information, such as class schedule or grade average, with a maximum of 3 interactions, taps or clicks, from the moment the application is opened.

- **Accessibility:** The interface must strictly comply with AgID guidelines and the WCAG 2.1 level AA standard, ensuring usability even for users with disabilities.

- **Learning Time:** A new user must be able to complete a classroom booking in less than 60 seconds without consulting the external guide.
