# OhMyUniversity! - RAD: 3.4.5 User Interface, Navigational Paths and Screen Mockups

This section describes the mid-fidelity user interface mockups created for OhMyUniversity!, capturing both web and mobile screen designs.
The design reference is published in Figma and includes the main navigational paths, layout approach, and screen structure for the proposed system.

## Figma Prototype

The mockups are available in Figma at the following link:

- https://www.figma.com/design/CMG7mltChXkrOmohGCZt6s/OMU?node-id=131-2005&p=f&t=G126IiVO7DAic6xw-0

This Figma file contains:

- Web screen designs for the desktop/browser version of OhMyUniversity!
- Mobile screen designs for the smartphone version of OhMyUniversity!
- Mid-fidelity interaction and navigation flows used to validate the proposed interface

## Purpose and Scope

This document links the existing interface mockups to the RAD artefacts. The Figma prototype is intended to:

- document the planned interface and navigation structure
- support review of the main user journeys and screen transitions
- provide a shared reference for designers, developers, and stakeholders
- compare both web and mobile design approaches in a single source

## Notes

The Figma file serves as the primary design reference for this section. Any future updates to the user interface mockups should be recorded here and reflected in the RAD documentation.
