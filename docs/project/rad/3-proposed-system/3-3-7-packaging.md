# 3.3.7 Packaging

The packaging requirements define how **OhMyUniversity!** must be distributed to end users and how the system must comply with legal constraints related to personal data. Since the platform handles academic and personal information, its release and data management policies must follow official distribution and privacy standards.

- **Distribution:** The software must be released through official channels, App Store and Google Play, and as a Web App accessible via browser without client-side configuration.

- **Legal Compliance:** Data processing must comply with GDPR, EU Regulation 2016/679. Sensitive data must be stored exclusively for the time necessary for the service's purpose.
