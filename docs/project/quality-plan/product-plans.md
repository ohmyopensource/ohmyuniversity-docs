# Product Plans

The current product plan focuses on the modules selected for the project delivery. The main objective is to provide a functional and coherent experience between the Web and Mobile applications, limiting the documentation and implementation scope to the features actually planned for the sprint.

The main modules included in the current release are:

- **Career, Didactics, and Student Administration Office:** visualization of academic career data, exams, grades, ECTS credits, averages, graduation base score, study plan, exam sessions, tuition fee status, and digital university badge.
- **Access to External University Services:** access to Moodle, institutional email, library services, and optimized consultation of exam sessions.
- **Organization and Logistics:** visualization of class timetables and management of an integrated calendar for academic commitments and deadlines.
- **Orientation and Future Planning:** support for enrollment procedures and access to course catalog information.

The expected deliverables are the working Web and Mobile applications, updated project documentation, the user manual, mock-ups where required, and the engineering documents updated according to the implemented scope.

The team also aims to keep the behavior of the Web and Mobile versions aligned, especially for shared functionalities such as authentication, academic career consultation, calendar management, and external-service access.
