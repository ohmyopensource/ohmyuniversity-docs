# User Manual - OhMyUniversity!

## Overview

This manual describes the features available in the Sprint 2 release of OhMyUniversity.

In this version, the student can:

* access the platform using university credentials;
* view academic career data, transcript, averages, CFU credits, and graduation base grade;
* view the study plan;
* consult exam sessions and bookings;
* view tuition fees and the digital badge;
* access external university services;
* consult class schedules and the calendar;
* use the orientation and enrollment guide.

Features not included in this release are not described in this manual.
